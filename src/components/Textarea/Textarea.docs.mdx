# Textarea

Multi-line text input for longer content.

## When to use
- Messages  
- Descriptions  
- Comments  
- Notes  

## Basic usage

```tsx
<Textarea placeholder="Write your messageâ€¦" />
```

## With FormField

Always wrap Textarea in FormField for:
- Label
- Helper text
- Error message
- Accessibility

```tsx
<FormField label="Message" helperText="Max 500 characters">
  <Textarea />
</FormField>
```
