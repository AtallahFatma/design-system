# Select

Dropdown for choosing one option from a list.

## When to use
- Country
- Status
- Category
- Role

## Basic usage

```tsx
<Select options={[{ label: "France", value: "fr" }]} />
````

## With FormFields
```tsx
<FormField label="Country" helperText="Choose your country">
  <Select options={[...]} />
</FormField>
```

## Accessibility
  - Must be wrapped in `FormField`
  - Uses `aria-invalid` on error
  - Uses `aria-describedby`
